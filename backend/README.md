# TJ POS Backend

Backend workspace for TJ POS.

## Package Manager

Use `pnpm` only.

```bash
pnpm install
pnpm dev
pnpm lint
pnpm typecheck
pnpm build
```

## Scope

This backend scaffold contains NestJS module files only. Business logic, real API
endpoints, database migrations, Redis jobs, and worker logic are intentionally
not implemented yet.
