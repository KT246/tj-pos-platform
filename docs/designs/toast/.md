Tên đúng cho dev sẽ là:

```text
TJ POS — Toast Notifications
```

hoặc:

```text
TJ POS — System Feedback Toasts
```

Mô tả:

```text
This screen defines the toast notification patterns used across TJ POS.

Use these toast components for:
- Success actions (Save completed, Order created, Payment received)
- Warning messages (Low stock, Expiring products, Pending actions)
- Error messages (Payment failed, Validation error, Network issue)
- Information messages (Data synced, Export started)

Toast placement:
- Top-right corner
- Auto dismiss after 3–5 seconds
- Manual close button
- Stack multiple notifications
- Mobile responsive

Types:
1. Success Toast
2. Warning Toast
3. Error Toast
4. Info Toast
```
